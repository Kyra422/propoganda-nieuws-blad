# Propaganda Nieuws Blad

Dit is de broncode voor de nieuwswebsite van onze Minecraft server.

## Hoe te bekijken

Deze website wordt gehost via GitHub Pages.

1. Ga naar de **Settings** (Instellingen) van deze repository.
2. Klik aan de linkerkant op **Pages**.
3. Onder "Build and deployment", kies bij **Source** voor "Deploy from a branch".
4. Selecteer onder **Branch** de optie `main` (of `master`) en de map `/ (root)`.
5. Klik op **Save**.

Na een paar minuten is je website live op `https://deegoo42.github.io/propoganda-nieuws-blad/`.

## Bewerken

- Bewerk `index.html` om het nieuws aan te passen.
- Bewerk `assets/css/style.css` om de stijl aan te passen.
